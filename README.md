### Project Overview
This project is an end-to-end machine learning pipeline designed to predict median house values in California. It demonstrates core software engineering principles applied to Data Science, including Object-Oriented Programming (OOP) and clean code practices.

### Tech Stack
* **Language:** Python 3.14
* **Libraries:** Pandas, Scikit-learn, NumPy
* **Version Control:** Git

### Project Structure
* `data/`: Contains the California Housing dataset.
* `src/`: Core logic including the `DataProcessor` class.
* `main.py`: Entry point for training and evaluating the model.
* `requirements.txt`: Managed dependencies.
